# 📦 TelegramScraper

> 🇬🇧 A collection of simple scripts built with **Telethon** for scraping, adding, and managing Telegram group/channel members.
>
> 🇮🇷 مجموعه‌ای از اسکریپت‌های ساده مبتنی بر **Telethon** برای استخراج، افزودن و مدیریت اعضای گروه‌ها و کانال‌های تلگرام.

---

# ✨ Features | امکانات

### 🇬🇧 English

- Scrape members from Telegram groups and supergroups
- Add members to groups or channels
- Send messages to scraped members
- Automatically remove blocked users from member lists
- Easy initial setup
- CSV-based workflow

### 🇮🇷 فارسی

- استخراج اعضای گروه‌ها و سوپرگروه‌های تلگرام
- افزودن اعضا به گروه یا کانال
- ارسال پیام به اعضای استخراج‌شده
- حذف خودکار کاربران بلاک‌شده از فایل اعضا
- راه‌اندازی آسان
- استفاده از فایل‌های CSV

---

# 📁 Project Structure | ساختار پروژه

| File | Description |
|------|-------------|
| `scraper.py` | Scrape members from a Telegram group/supergroup and save them to `members.csv`. |
| `adder.py` | Add members to Telegram groups using predefined settings. |
| `add2channel.py` | Add members from `members.csv` to a Telegram channel or supergroup. |
| `add2channel_clean.py` | Same as `add2channel.py` but removes blocked users from `members.csv`. |
| `add2group.py` | Add members to a selected Telegram group. |
| `smsbot.py` | Send messages to users listed in `members.csv`. |
| `setup.py` | Initial setup wizard for creating `config.data`. |
| `config.data` | Stores `api_id`, `api_hash`, and `phone`. **Do not publish this file.** |
| `members.csv` | Member database generated by `scraper.py`. |
| `LICENSE` | Project license. |

# 🚀 Quick Start | شروع سریع

## 1️⃣ Initial Setup
### راه‌اندازی اولیه

```bash
python3 setup.py
```

---

## 2️⃣ Scrape Members
### استخراج اعضا

```bash
python3 scraper.py
```

The extracted members will be saved to:

```
members.csv
```

---

## 3️⃣ Add Members
### افزودن اعضا

### Add to Group

```bash
python3 add2group.py members.csv
```

### Add to Channel

```bash
python3 add2channel.py members.csv
```

### Clean Blocked Users While Adding

```bash
python3 add2channel_clean.py members.csv
```

---

## 4️⃣ Send Messages
### ارسال پیام

```bash
python3 smsbot.py members.csv
```

---

# 🔐 Configuration | تنظیمات

Run the setup wizard:

```bash
python3 setup.py
```

You'll be asked for:

- `api_id`
- `api_hash`
- `phone number`

The information will be stored in:

```
config.data
```

> **Do NOT upload this file to GitHub.**
>
> **این فایل را داخل GitHub منتشر نکنید.**

# 📂 Output Files | فایل‌های خروجی

members.csv
