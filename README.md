# TelegramScraper

مجموعه‌ای از اسکریپت‌های ساده برای کار با اعضای گروه‌ها و کانال‌های تلگرام با استفاده از کتابخانه Telethon.

پیش‌نیازها

- Python 3.8 یا بالاتر
- نصب شده: `telethon` (اسکریپت `setup.py` تلاش می‌کند آن را نصب کند)

فایل‌های مهم و توضیحات

- `scraper.py`: استخراج اعضای یک گروه/سوپرگروه و ذخیره در `members.csv`.
- `adder.py`: افزودن اعضا به گروه‌ها (نسخه‌ای با تنظیمات ثابت برای اجرا). پیش از اجرا `members.csv` را آماده کنید.
- `add2channel.py`: افزودن اعضا به یک کانال یا سوپرگروه (با انتخاب از `members.csv`).
- `add2channel_clean.py`: مشابه `add2channel.py` اما هنگام مواجهه با کاربران بلاک‌شده، آن‌ها را از فایل اعضا حذف می‌کند.
- `add2group.py`: افزودن اعضا به گروه‌ها (با انتخاب گروه هدف از لیست گفتگوها).
- `smsbot.py`: ارسال پیام به لیستی از کاربران (از `members.csv`).
- `setup.py`: راه‌اندازی اولیه؛ ایجاد فایل `config.data` و دریافت `api_id`، `api_hash` و شماره تلفن.
- `config.data`: فایل پیکربندی حاوی مقادیر `api_id`, `api_hash` و `phone` (در صورت نیاز روی مخزن منتشر نکنید).
- `members.csv`: خروجی `scraper.py` یا ورودی برای ابزارهای افزودن/ارسال پیام (نیست در مخزن اگر حذف شده باشد).
- `LICENSE`: متن حقوقی پروژه.

نکات امنیتی و انتشار

- فایل‌های جلسه (`*.session`) و متادیتای macOS پاک شده‌اند تا اطلاعات حساس منتشر نشوند.
- طبق درخواست، تمام کامنت‌های داخل کدها حذف شده‌اند تا فایل‌ها برای انتشار تمیز باشند. عملکرد کدها تغییری نکرده مگر حذف توضیحات متنی؛ اگر می‌خواهید توضیحات مستندسازی شده را به صورت جداگانه اضافه کنم، اطلاع دهید.

نحوه استفاده سریع

1. اجرای تنظیم اولیه:

```bash
python3 setup.py
```

2. استخراج اعضا از یک گروه:

```bash
python3 scraper.py
```

3. افزودن اعضا به گروه یا کانال (مثال):

```bash
python3 add2group.py members.csv
python3 add2channel.py members.csv
```

4. ارسال پیام به اعضا:

```bash
python3 smsbot.py members.csv
```
